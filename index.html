HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1, h2, h3, etc. element represents all the titles and the order in which they are assembled.
  • The lu element creates an unordered list of items.
  • The li element is used for making bullet points.
  • The elements are enclosed with the initial command, the content inside and a "/" with the command right after.

CSS

  • CSS is a language for specifying how documents are presented to users — how they are styled, laid out, etc.
  • A font size indicates the size you want the font to be in your html.
  • The height element decides how many pixels of height an element will have.
  • You can choose the font that you would like to use in the HTML.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • The git status command gives you the status of which branch you are located in.
  • The git commit command lets you commit any changes that you have made to a file in your code.
  • The git config -global user.name command lets you set a name that is identifiable for credit when review version history.
  • The git config _global user.email command lets you set an email address that will be associated with each history marker.
  • The git checkout command lets you switch to another branch and check it out into your working directory.
  • The git log command can show all commits in the current branch's history.

JavaScript

  • It is a scripting or programming language that allows you to implement complex features on web pages.
  • The let command assigns a value to a variable.
  • The const command assigns many values to a variable.
  • It is not imited to webpages, it can also be used for mobile aplications.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The randomTopic function allows you to randomly select a variable so that every time you refresh a page it gives you a different outcome.
